# 📢 Facebook Ads Capstone Project – Webify

## 📖 Project Overview

This repository contains my Facebook Ads Capstone Project, developed as part of my Digital Marketing coursework. The project demonstrates the planning, setup, and analysis of a Facebook advertising campaign for **Webify**, a web design and digital marketing agency.

The objective of this project is to showcase the complete process of creating a Facebook Ads campaign—from defining business goals and identifying the target audience to budgeting, ad creation, campaign setup, and performance evaluation.
